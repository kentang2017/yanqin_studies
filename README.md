# 演禽/禽遁廿八宿研究書目
搜羅所有與演禽有關的書目。演禽，始於二十八宿占星玩意，究其源起，可追溯秦漢時期出土文物放馬灘簡的《三十六禽占》，而真正出現 "演禽"二字則始於宋代，據宋史收錄有關演禽術數書籍，已結合了大六壬，然而蒙古人打主中原，大量經書因戰亂燒毀或散佚。後來禽遁盛行於明清，深得兵家學者所愛。
